# ofxFFmpegRecorder

A small class to utilize the video recording functionality of `ffmpeg`.

# Features

- Record video
- Record audio
- Save thumbnail from a video
- Record custom video by adding `ofPixels`
- Pause the custom video recording

# How to Use

See the examples.

# Dependencies

ofxFFmpegRecorder depends openFrameworks; ffmpeg libs for macos and visual studio are included. ffmpeg can be installed via apt on linux.
